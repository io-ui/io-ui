# 安装

## npm 安装

```bash
npm install io-ui
```

## yarn 安装

```bash
yarn add io-ui
```

## CDN
目前可以通过 unpkg.com/io-ui 获取到最新版本的资源，在页面上引入 js 和 css 文件即可开始使用。

```html
<!-- 引入样式 -->
<link rel="stylesheet" href="https://unpkg.com/browse/io-ui/package/main.css">
<!-- 引入组件库 -->
<script src="https://unpkg.com/browse/io-ui/package/io-ui.js"></script>
```
